# SkillSwap

SkillSwap is a modern full-stack application for college students to discover ideal teammates for academic projects, hackathons, research, and startup ideas using AI-driven skill matching.

## Features
- Responsive React + Tailwind UI with glassmorphism design
- AI-style backend endpoints for match recommendations, project ideas, and profile insights
- Mock student profiles, project hub, discovery, and chat assistant UI
- Express backend with reusable API routes

## Install

### Frontend
```bash
cd SkillSwap/frontend
npm install
npm start
```

### Backend
```bash
cd SkillSwap/backend
npm install
npm run dev
```

## Notes
- The current build is a hackathon-ready prototype with AI-style workflows using mock data.
- Replace `REACT_APP_API_URL` and backend environment variables for real deployment.
- Use MongoDB Atlas and OpenAI or Gemini APIs for production-level AI and data storage.
